# Sun Oaks Brand House

A comprehensive brand house and resource center for Sun Oaks Wellness & Family Tennis, Fitness, and Pool Club in Redding, California.

## Project Overview

This brand house serves as the central hub for Sun Oaks' brand identity, guidelines, and marketing resources. It includes the main brand house interface and specialized resource pages for different aspects of the business.

## Getting Started

### Viewing the Brand House

1. Open `index.html` in a web browser to view the main brand house
2. Navigate through the chapters to explore brand resources
3. Click on "01 Brand Foundation" to view the brand foundation chapter

### Navigation Structure

- **Homepage (`index.html`)**: Main brand house with 10 chapters
  - Links to Brand Foundation page
  - Contains brand overview, manifesto, and service architecture

- **Pages Directory**: Contains detailed resource pages
  - All pages have "Back" navigation to return to the homepage
  - Each page maintains consistent design language with the brand house

## Brand Elements

### Colors
- **Sun Yellow**: #F7B500 (Primary brand color)
- **Pool Teal**: #0891A8 (Secondary brand color)
- **Cream Background**: #FAF9F7
- **Black**: #0D0D0C

### Typography
- **Display Font**: Jost (Futura alternative)
- **Editorial Font**: Newsreader (Serif)

### Service Lines
Sun Oaks offers seven distinct service experiences:
1. Fitness (120+ Classes)
2. Tennis (10 Courts)
3. Pickleball (4 Dedicated Courts)
4. Aquatics (3 Pools)
5. Kids Programs
6. Spa Services
7. Event Venue Rentals
